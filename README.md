# HEX
Hex is a tool used for generating random stuff

# Download
```
git clone https://github.com/daniscode1/HEX

cd HEX
```

# Usage
```
HEX HELP
python3 hex.py [commands: -help, -version, -random-num, -random-country, -random-name, -random-gender, -profile]
```
